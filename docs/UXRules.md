# UX Rules

## Core UX
- User harus bisa mulai latihan dari Home dalam satu tap.
- Daily Qira adalah entry point utama.
- Feedback harus selalu memberi next action.
- Jangan membuat user merasa dihakimi.
- Jangan menyembunyikan Arabic text di antara dekorasi.

## Screen Rules
### Home
- Daily Qira selalu berada di viewport awal.
- AI greeting harus terasa personal.
- Weekly snapshot hanya pendukung, bukan pusat.

### Practice
- Reference audio dan recording harus mudah ditemukan.
- State recording harus sangat jelas.
- Jangan menaruh terlalu banyak statistik di flow latihan.

### Evaluation
- Score harus ditemani explanation.
- Highlight harus memakai semantic color yang konsisten.
- Primary action setelah result: retry atau next.

### Insight
- Visualisasi harus menjawab: "apa yang perlu gue latih?"
- Letter mastery lebih penting daripada angka kosong.

### Profile
- Profile tidak boleh menjadi tempat fitur acak.
- Profile berisi identitas, preferences, dan learning summary.

## Empty State
Empty state harus memberi action berikutnya.

## Error State
Error state harus:
- Menjelaskan masalah dengan bahasa sederhana.
- Memberi retry jika memungkinkan.
- Tidak menampilkan stack trace.
