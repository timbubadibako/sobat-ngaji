# Definition of Done

Sebuah feature dianggap selesai jika semua checklist berikut terpenuhi.

## Product
- Requirement feature terpenuhi.
- Acceptance criteria terpenuhi.
- Copy sesuai tone Sobat Ngaji.
- Tidak ada klaim AI yang absolut.

## Architecture
- Feature mengikuti folder structure.
- Business logic tidak berada di widget.
- API/local storage lewat repository/service.
- Provider dependency jelas.
- Tidak ada duplicate component.

## Design
- UI mengikuti DesignGuideline.
- Palette tidak berubah.
- Spacing, radius, typography, dan state konsisten.
- Loading, empty, error, dan success state tersedia jika relevan.

## Code Quality
- Flutter analyze pass.
- Format pass.
- Tidak ada dead code.
- Public class penting terdokumentasi.

## Testing
- Unit test untuk controller/repository penting.
- Widget test untuk UI state penting.
- Manual QA untuk flow utama.

## Handoff
- Files created/modified disebutkan.
- Perubahan dijelaskan singkat.
- Risiko atau asumsi disebutkan.
