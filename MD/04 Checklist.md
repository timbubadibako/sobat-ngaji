# Checklist

## Checklist Seleksi Awal / Seleksi Nasional ITDP

- [ ] Proposal final sudah siap dalam format PDF
- [ ] Ukuran proposal maksimal 8 MB
- [ ] Nama file proposal sesuai format resmi
- [ ] Subjek unggahan proposal sesuai format resmi
- [ ] Cover mengikuti Lampiran 1
- [ ] Lembar pengesahan mengikuti Lampiran 2
- [ ] Abstrak sudah 150-200 kata
- [ ] Latar belakang sudah final
- [ ] Tujuan dan manfaat sudah final
- [ ] Metode pengembangan produk sudah final
- [ ] Analisis fungsional sudah final
- [ ] Desain produk teknologi digital sudah final
- [ ] Pembahasan implementasi, validasi, demo, dan dampak sudah final
- [ ] Tautan video sudah dimasukkan ke proposal
- [ ] Daftar pustaka sudah final

## Checklist Prototype dan Demo

- [ ] Prototype minimal 50% sudah siap
- [ ] Alur `pilih ayat -> rekam -> proses -> hasil evaluasi` bisa didemokan
- [ ] Tampilan hasil evaluasi sudah bisa ditunjukkan
- [ ] Ada mockup atau screenshot pendukung
- [ ] Ada flow penggunaan aplikasi
- [ ] Demo 3 menit sudah disusun
- [ ] Subtitle video sudah disiapkan
- [ ] Intro video sudah disiapkan
- [ ] Format video MP4 720p
- [ ] Judul video YouTube sesuai format resmi

## Checklist Administratif

- [ ] Tim terdiri dari 3-4 mahasiswa aktif
- [ ] Dosen pendamping sudah fix
- [ ] KTM semua peserta siap
- [ ] Kode PT sudah benar
- [ ] Nama tim sudah benar
- [ ] Berita acara seleksi internal PT siap
- [ ] Surat pernyataan orisinalitas atau keterangan karya siap
- [ ] Surat transparansi AI siap
- [ ] Similarity report original report siap
- [ ] Similarity report maksimal 25%

## Checklist Kualitas Proposal

- [ ] Masalah pendidikan yang diangkat jelas
- [ ] Fokus karya tetap pada koreksi bacaan, bukan hafalan
- [ ] Istilah `Arabic Automatic Speech Recognition (ASR)` konsisten
- [ ] Scope MVP tidak melebar
- [ ] Tidak ada klaim yang berlebihan
- [ ] Produk diposisikan sebagai alat bantu, bukan pengganti guru
- [ ] Ada nilai kebaruan yang jelas
- [ ] Ada rencana validasi pengguna
- [ ] Ada dampak pendidikan yang jelas

## Format Resmi yang Harus Diikuti

### Nama file proposal

`LIDM 2026 - ITDP - Kode PT - Nama Tim - Proposal.pdf`

### Subjek unggahan proposal

`LIDM 2026 - ITDP - Kode PT - Nama Tim - Proposal`

### Judul video YouTube

`LIDM 2026 - ITDP - Kode PT - Nama Tim - Judul Karya - Proposal`
